## 🤖 AI Chat Assistant

This site includes an interactive AI chat widget that allows visitors to ask questions about my background, experience, and qualifications. The AI is designed to:

- Answer questions about my work history and experience
- Discuss my technical skills and expertise
- Share my education and achievements
- Provide contact information

The chat widget only responds to questions related to my resume and professional background.

## 🚀 Deployment

This site is deployed using GitHub Pages. The static files are served directly from this repository.

### Manual Deployment

To deploy manually:

1. Clone this repository
2. Go to Settings → Pages
3. Select "main" branch as source
4. Save and wait for deployment
